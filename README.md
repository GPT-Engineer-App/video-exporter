# video-exporter

An website that allows users to upload a youtube video by url and the website can grab the title thumbnail and transcript of the video and the user has the option to export these videos as a word,text and or pdf file

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/video-exporter.git
cd video-exporter
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
